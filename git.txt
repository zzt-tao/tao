git init 初始化仓库
git status 查看仓库状态
git add 文件名 添加文件到本地仓库

git config --global user.email 我的邮箱
git config --global user.name 我的名字

git commit -m '注释' 提交文件

git remote 查看已添加的远程仓库
git remote add origin '仓库地址'

git push '远程仓库的别名'(默认为ori) master

ssh-keygen  -t rsa -C '我的邮箱'

cat id_rsa.pub
将该文件的内容（公钥）放入设置的ssh and GPG keys
